# FCAI Student Activities Repository (SAR)

## Project Overview
The Student Activities Repository (SAR) is a front-end web application developed for the Faculty of Computers and Artificial Intelligence (FCAI). The project provides students with an interactive platform to explore, register for, and manage extracurricular activities organized within the faculty.

SAR centralizes student activities in one system, making it easier for students to participate, track events, and stay connected with campus activities.

## Features
- **Homepage**
  - Central navigation point for accessing all system pages

- **Registration Page**
  - Allows students to register for the system or specific activities using a form

- **Booking Page**
  - Enables students to book available events and confirm participation

- **Calendar Page**
  - Displays upcoming events in a dynamic calendar
  - Allows adding and removing events
  - Uses browser local storage to save event data

- **History Page**
  - Shows past activities
  - Supports filtering by activity type
  - Provides access to attached event materials

- **Contact Us Page**
  - Displays contact information
  - Allows users to submit inquiries or feedback

## Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- Local Storage API
